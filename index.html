Frequently Asked Questions

Common questions about ClashFox and Mihomo

Why does ClashFox need System Extensions approval?

ClashFox uses macOS System Extensions to manage system-level proxy settings. Since macOS Catalina (10.15), Apple requires user approval for third-party system extensions to ensure security. You can approve this in System Preferences > Security & Privacy.

How do I resolve kernel conflict issues?

If you're experiencing conflicts, make sure you don't have other Mihomo-based applications running simultaneously (like ClashX Pro or Surge). Each application needs its own Mihomo instance. Close other proxy apps before starting ClashFox.

Where can I find the Mihomo configuration file?

ClashFox looks for Mihomo configuration at ~/.config/mihomo/config.yaml by default. You can also specify a custom path in ClashFox preferences. Make sure your configuration file follows the Mihomo format.

Does ClashFox support proxy subscription links?

Yes! ClashFox supports importing proxy configurations via subscription URLs. Go to Configuration > Import from URL and paste your subscription link. ClashFox will automatically fetch and update your proxy list.

How to enable LaunchDaemon for auto-start?

ClashFox can install a LaunchDaemon helper for automatic startup. Go to Settings > General > Launch at Login. This allows ClashFox to start automatically when you log in to macOS.
